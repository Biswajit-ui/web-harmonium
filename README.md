# Web Harmonium
A real harmonium sample engine built entirely in the browser.

**Live Demo:** https://biswajit-ui.github.io/web-harmonium/

## Features
- Real harmonium audio sample
- Volume, Reverb, Transpose, Octave controls
- Record & playback your sessions
- MIDI device support
- Light / Dark mode
- Mobile friendly
